# Mahfil App Testing & Bug Report

## 📌 Overview
This repository contains test cases and bug reports for the Mahfil APK application. The purpose of this project is to ensure application quality through systematic testing and issue tracking.

## 🧪 Testing Scope
- Functional Testing
- UI/UX Validation
- Bug Identification & Reporting



## 📊 Test Case Details
Each test case includes:
- Test Case ID
- Module Name
- Test Description
- Expected Result
- Actual Result
- Status (Pass/Fail)

## 🐞 Bug Report Details
Each bug includes:
- Bug ID
- Description
- Severity
- Steps to Reproduce
- Expected vs Actual Result
- Status (Open/Closed)
